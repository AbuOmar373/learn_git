إنشاء حساب 
git config --global user.name "mamri"
git config --global user.email "alazdi2007@gmail.com"

التأكد من البيانات:
git config --list

موقع المسار الحالي:
pwd

إنشاء مشروع جديد:
git init

إظافة جميع الملفات:
git add .

إضافة ملف لوحده:
git add README.md

إضافة تعليق:
git commit -m "first commit"

عرض التعليقات السابقة:
git log

عرض الملفات غير المظافة:
git status

استعراض عمل كاتب محدد:
git log --author "mamri"


====== github =====

للتحميل أول مره نذهب للفولدر ثم:
git clone https://github.com/ww5z/archives.git

لإعادة لرفع الملفات بعد التعديل:
git add .
git commit -m "big updates"
git push origin master
Or (git push -u origin master)

إذا حدث آخر وجلب البيانات بعد التعديل
git pull origin master

= لستعادة ملف لوحده
استعراض:
git diff chapter3.txt
استعادة:
git checkout chapter3.txt
